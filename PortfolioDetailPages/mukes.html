Mukes

Android


Google Play Source Code

This application gets menus for all Leijona Catering restaurants on garrison areas from their RSS-feed and shows them on a simple and easy to read list. I started making this app when I was in the military and finished it when I got out from there. I developed this app to be cross-platform using C#-language and Xamarin for Visual Studio. The Android version is fully functional and available to download in the Google Play store. It would be easy to make a functional iOS version because I used classes and I can use the same base code on both Android and iOS. I’m planning to do the iOS version ready soon when I get other iOS apps ready for publication and I get the developer account for App Store.

The source code for this app is available in my GitHub-page.
